# Xtreme Collision Website

Local recreation of the [Xtreme Collision](https://xcollision.vercel.app/) marketing site — a Next.js App Router + Tailwind CSS landing page for the Carrollton, TX auto body shop.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router)
- React 19
- Tailwind CSS v4
- Lucide icons
- Geist + Oswald fonts
